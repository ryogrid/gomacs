## Codebase Patterns
- Buffer is stored as `[][]rune` (Lines field), one slice per line
- Buffer tracks cursor via `CursorR`/`CursorC` fields
- Mark/Region: `MarkR`, `MarkC`, `MarkActive` on Buffer; `SetMark()`, `DeactivateMark()`, `KillRegion()`, `CopyRegion()`
- C-SPC currently handled at main.go:287 as `tcell.KeyCtrlSpace`
- Key event loop in main.go uses `switch ev.Key()` for dispatch
- tcell.KeyCtrlSpace = 64 (iota + 64), tcell.KeyNUL = 0 (iota) — different constants
- tcell input processor maps raw byte 0x00 to KeyCtrlSpace via `KeyCtrlSpace + Key(r)`
- Windows Terminal on WSL2 may not deliver NUL byte for Ctrl+Space
